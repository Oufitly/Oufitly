# 👗 Outfitly – AI Outfit Planner App

Outfitly is a smart fashion assistant app that helps users decide what to wear every day using their own wardrobe. Designed especially for young users (18–25), it simplifies daily outfit planning with a clean and modern interface.

---

## 🚀 Features

* 👕 Upload and manage your wardrobe
* 🤖 Smart outfit suggestions
* 📅 Daily outfit planner
* 🎯 Occasion-based styling (casual, party, college)
* 💡 Budget-friendly fashion tips
* 🔐 Simple login system

---

## 📱 Screens

* Login / Signup
* Home (Outfit suggestions)

* Planner (coming soon)

---

## 🛠️ Tech Stack

* React Native
* Expo
* Expo Router
* JavaScript

---

## 📦 Installation & Setup

Follow these steps to run the app locally:

```bash
npm install
npm start
```

Then scan the QR code using Expo Go app on your phone 📱
